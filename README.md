# 🚚 Transport & Logistics Dashboard Project

## 📌 Project Overview

The **Transport & Logistics Dashboard** project is a comprehensive data analytics solution designed to monitor, evaluate, and optimize logistics operations. Built using Power BI, this project transforms raw transportation and shipment data into meaningful insights through interactive visualizations. It enables stakeholders to make data-driven decisions by identifying inefficiencies, tracking performance, and improving overall supply chain effectiveness.

The dashboard integrates multiple datasets to provide a unified view of logistics operations, including customer behavior, shipment trends, route efficiency, and transport mode performance. It serves as a decision-support tool for operations managers, analysts, and business leaders.

---

## 📊 Dataset Description

The project utilizes structured datasets that represent key components of a logistics system:

* **Customers Dataset**
  Contains detailed customer information such as customer ID, type (B2B or B2C), and geographic region. This dataset helps in understanding customer distribution and demand patterns.

* **Shipments Dataset**
  Includes critical shipment-related data such as shipment ID, weight, cost, delivery status, delivery time, and delays. This is the core dataset used for performance and efficiency analysis.

* **Routes Dataset**
  Provides origin and destination details along with distance metrics. This dataset is essential for evaluating route efficiency and transportation planning.

* **Transport Modes Dataset**
  Defines different modes of transportation such as Air, Rail, Road (Truck), and Sea. It helps in comparing cost, speed, and reliability across transport options.

---

## 📈 Key Insights & Analysis

The dashboard reveals several important business insights:

* **Delivery Performance**
  A significant portion of shipments experience delays, indicating operational bottlenecks. Monitoring delivery status helps in improving service reliability.

* **Cost Analysis**
  Transportation costs vary widely depending on distance, shipment weight, and transport mode. Air transport tends to be faster but more expensive, while rail and road offer cost-effective alternatives.

* **Route Efficiency**
  Certain routes consistently show higher delivery times and costs, suggesting the need for route optimization or alternative planning.

* **Transport Mode Comparison**
  Each transport mode has trade-offs between cost and delivery speed. The dashboard helps in selecting the most efficient mode based on business needs.

* **Customer Segmentation**
  B2B customers often contribute to bulk shipments, while B2C customers show higher frequency but lower volume. Regional analysis highlights high-demand areas.

* **Priority Impact**
  Express shipments reduce delivery time but increase cost, while standard shipments are more economical but slower.

## 📊 Dashboard Features

The Power BI dashboard includes the following interactive components:

* **Shipment Status Overview**
  Visual breakdown of delivered, pending, and delayed shipments.

* **Cost & Weight Analysis**
  Insights into how shipment weight affects transportation cost.

* **Route Performance Visualization**
  Identification of high-performing and underperforming routes.

* **Delivery Time Analysis**
  Tracking average delivery time and delay patterns.

* **Transport Mode Comparison**
  Comparative analysis of Air, Rail, Road, and Sea transport.

* **Customer Insights**
  Segmentation by customer type and region for better demand understanding.

* **Filters & Drill-Downs**
  Interactive filters for region, transport mode, and shipment status to enable deeper analysis.
